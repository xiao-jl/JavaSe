# JavaSE

## 项目结构概览

```text

src/
├── h666/-----------------------# 综合项目模块
│   └── com/
│       ├── denglu/-------------# 登录功能相关实现
│       ├── jiaoxue/------------# 教学/演示相关代码
│       └── xiao/---------------# 个人练习模块
│
└── xuexi/----------------------# Java 基础学习模块
    └── xiao/
        ├── arraylist/----------# ArrayList
        ├── duandian/-----------# 断点
        ├── duixiang/-----------# 面向对象基础
        ├── fangfa/-------------# 方法的定义与调用
        ├── jiben/--------------# Java 基本语法/数据类型
        ├── kongzhi/------------# 流程控制
        ├── liang/--------------# 变量与常量
        ├── random/------------ # Random
        ├── scanner/------------# Scanner
        ├── string/------------ # String
        ├── yinyongshujuleixing/# 引用数据类型进阶
        ├── yufalianxi/---------# 语法练习
        ├── zongjie/------------# 阶段性知识总结
        └── Zu/-----------------# 数组
```
