
关键字

Java JDK8 关键字



保留关键字:（50个）

    1. 基本数据类型关键字（8个）
        整数:(默认使用:int)
            byte (8位有符号整数，范围 -128 到 127)
            short (16位有符号整数，范围 -32,768 到 32,767)
            int (32位有符号整数，范围 -2,147,483,648 到 2,147,483,647)
            long (64位有符号整数，范围 -2^63 到 2^63-1)       (值的末尾加 L)
        浮点数:(默认使用:double)
            float (32位单精度浮点数，范围 1.4E-45 到 3.4E38)       (值的末尾加 F)
            double (64位双精度浮点数，精度高于 float，范围 4.9E-324 到 1.8E308)
        字符:
            char (16位Unicode字符，用于表示单个字符，如 'A','1','@','我')      (char 类型本身就是数值类型,16位ASCII表)
        布尔:
            boolean (只有两个值：true 和 false)

    2. 访问控制关键字（3个）
            private (访问控制：私有，只能在声明它的类内部访问)
            protected (访问控制：受保护，可以在声明它的类及其子类中访问，同包内其他类也可以访问)
            public (访问控制：公共，可以在任何地方访问)

    3. 类、接口和枚举相关关键字（6个）
            class (用于声明一个类，类是面向对象编程的基本构造块)
            interface (用于声明一个接口，接口定义了一组抽象方法，类可以实现多个接口)
            enum (用于声明一个枚举类型，枚举是一种特殊的类，用于定义一组固定的常量)
            extends (用于继承一个类或实现一个接口，表示当前类继承自指定的父类)
            implements (用于实现一个或多个接口，表示当前类实现了指定的接口)
            abstract (用于声明抽象类或抽象方法，抽象类不能被实例化，抽象方法没有实现)

    4. 方法相关关键字（5个）
            void (用于声明方法没有返回值)
            static (用于声明静态变量或静态方法，静态成员属于类本身，而不是类的实例)
            final (用于声明常量、最终类或最终方法，最终变量不能被修改，最终类不能被继承，最终方法不能被重写)
            synchronized (用于声明同步方法或同步代码块，确保在同一时间只有一个线程可以访问)
            native (用于声明本地方法，本地方法是由其他语言（如C或C++）实现的方法)

    5. 控制流关键字（11个）
            if (用于条件判断，如果条件为真，则执行相应的代码块)
            else (用于 if 条件不满足时执行的代码块)
            switch (用于多分支选择，根据变量的值选择不同的执行路径)
            case (用于 switch 语句中的分支条件)
            default (用于 switch 语句中的默认分支，当没有匹配的 case 时执行)
            for (用于循环结构，通常用于已知循环次数的情况)
            while (用于循环结构，通常用于未知循环次数的情况)
            do (与 while 配合使用，用于至少执行一次的循环)
            break (用于终止循环或 switch 语句)
            continue (用于跳过当前循环的剩余部分，继续下一次循环)
            return (用于从方法中返回值或终止方法执行)

    6. 异常处理关键字（5个）
            try (用于包裹可能抛出异常的代码块)
            catch (用于捕获并处理异常)
            finally (用于包裹无论是否发生异常都执行的代码块)
            throw (用于显式抛出异常)
            throws (用于声明方法可能抛出的异常)

    7. 其他关键字（12个）
            new (用于创建对象或数组)
            this (指向当前对象的引用，用于引用当前对象的成员)    (this不能用于static方法中)
            super (用于调用父类的构造方法、属性或方法)
            instanceof (用于检查对象是否是特定类的实例)
            assert (用于断言，通常用于调试和测试)
            volatile (用于声明易变变量，确保多线程环境下的可见性)
            transient (用于声明序列化时忽略的变量)
            package (用于声明当前文件的包名)
            import (用于导入其他包中的类或接口)
            goto (保留关键字，目前未使用)
            const (保留关键字，目前未使用)
            strictfp (用于声明浮点运算遵循 IEEE 754 标准)


非保留关键字:

    1.引用数据类型
            String (字符串数据类型关键字)
    2.面量
        布尔:
            true (真)
            false (假)
        引用:
            null (空)

    3.其他
            length (遍历数组的长度)
            equals (比较两个值是否相等)
